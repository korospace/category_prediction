## Requirements

- php 7.3 or higher
- [composer 2.3.4](https://getcomposer.org/)
- mysql

## Setup

<ol>
    <li> Clone <b>or</b> Download as ZIP
    <li> Change Directory to root project </li>
    <li> Create database <b>db_pemketir</b> </li>
    <li> run:</li>
</ol>

```
composer install

php spark migrate

php spark serve -port 8085
```
